# Portfolio

Source for my personal portfolio site at work.czvck.com.

Self-hosted on a Raspberry Pi 400 running nginx + Flask. 
Frontend is vanilla HTML/CSS/JS with Chart.js for data visualization. 
Backend is a Flask API backed by SQLite.

## Stack
- nginx (reverse proxy)
- Flask (API)
- SQLite (database)
- Chart.js (visualization)
- Vanilla JS/HTML/CSS
